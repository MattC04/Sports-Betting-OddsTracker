# STEP-BY-STEP GUIDE - START HERE

Follow these steps exactly in order. Do not skip any step.

## STEP 1: Download All Files

Make sure you have downloaded ALL these files to the SAME folder:

- setup_and_run.py
- dashboard.py
- dashboard.html
- odds_api_scraper.py
- player_props_scraper.py
- requirements.txt

Check by running:
```
dir        (Windows)
ls -la     (Mac/Linux)
```

You should see all 6 files listed.

---

## STEP 2: Get Your API Key

1. Go to: https://the-odds-api.com/
2. Sign up for a free account
3. Copy your API key (it looks like: abc123def456...)
4. Keep it ready for the next step

---

## STEP 3: Open Terminal/Command Prompt

**Windows:**
- Press Windows key
- Type "cmd"
- Press Enter

**Mac:**
- Press Cmd+Space
- Type "terminal"
- Press Enter

**Linux:**
- Press Ctrl+Alt+T

---

## STEP 4: Navigate to Your Project Folder

Type this command (replace with your actual path):

**Windows:**
```
cd C:\Users\YourName\Downloads\sports-betting-dashboard
```

**Mac/Linux:**
```
cd /Users/YourName/Downloads/sports-betting-dashboard
```

Verify you're in the right place:
```
dir        (Windows)
ls -la     (Mac/Linux)
```

You should see all your files listed.

---

## STEP 5: Set Your API Key

**IMPORTANT:** Replace 'your_api_key_here' with your ACTUAL API key from Step 2

**Windows Command Prompt:**
```
set ODDS_API_KEY=your_api_key_here
```

**Windows PowerShell:**
```
$env:ODDS_API_KEY='your_api_key_here'
```

**Mac/Linux:**
```
export ODDS_API_KEY='your_api_key_here'
```

Verify it's set:

**Windows:**
```
echo %ODDS_API_KEY%
```

**Mac/Linux:**
```
echo $ODDS_API_KEY
```

You should see your API key printed. If not, try the command again.

---

## STEP 6: Run the Automated Setup

Type this command:
```
python setup_and_run.py
```

This script will:
1. Check your Python version
2. Check your API key
3. Install required packages
4. Check all files are present
5. Create data directory
6. Test API connection
7. Fetch player props data
8. Ask if you want to start the dashboard

**What to expect:**
- You'll see "STEP 1", "STEP 2", etc. messages
- Each step should show "SUCCESS"
- If any step fails, it will tell you what's wrong
- The whole process takes 1-2 minutes

**If it asks to start the dashboard:**
- Type 'y' and press Enter
- Wait for "Running on http://0.0.0.0:5000"
- Open your browser to: http://localhost:5000

---

## STEP 7: View Your Dashboard

Open your web browser and go to:
```
http://localhost:5000
```

You should see:
- Today's games
- Player props
- Statistics

---

## IF SOMETHING GOES WRONG

### Error: "python is not recognized"

**Try:**
```
python3 setup_and_run.py
```

If that doesn't work, Python is not installed. Download from: https://python.org

### Error: "ODDS_API_KEY not set"

You didn't complete Step 5 correctly. Go back and do Step 5 again.

### Error: "No games today"

NBA might not have games today. Try:
```
python setup_and_run.py
```
And change sport to NFL in the script, or wait for a game day.

### Error: "Cannot connect to API"

Check your internet connection. Make sure you can access: https://the-odds-api.com/

### Dashboard won't open

Make sure you see "Running on http://0.0.0.0:5000" in your terminal.
Then try: http://127.0.0.1:5000 in your browser instead.

## MANUAL METHOD (If Automated Setup Fails)

If setup_and_run.py doesn't work, do this manually:

### 1. Install packages
```
pip install Flask flask-cors requests pandas
```

### 2. Set API key (see Step 5)

### 3. Create data folder
```
mkdir data
```

### 4. Run this Python code

Create a file called `fetch_data.py` with this content:

```python
import os
import requests
import sqlite3
from datetime import datetime

api_key = os.getenv('ODDS_API_KEY')
if not api_key:
    print("ERROR: Set API key first")
    exit(1)

print("Fetching events...")
response = requests.get(
    "https://api.the-odds-api.com/v4/sports/basketball_nba/events",
    params={'apiKey': api_key}
)

events = response.json()
print(f"Found {len(events)} events")

if events:
    event_id = events[0]['id']
    print(f"Fetching props for event {event_id}...")
    
    response = requests.get(
        f"https://api.the-odds-api.com/v4/sports/basketball_nba/events/{event_id}/odds",
        params={
            'apiKey': api_key,
            'regions': 'us',
            'markets': 'player_points'
        }
    )
    
    data = response.json()
    print(f"Got data, saving to database...")
    
    conn = sqlite3.connect('data/player_props.db')
    cursor = conn.cursor()
    
    cursor.execute("""
        CREATE TABLE IF NOT EXISTS player_props (
            id INTEGER PRIMARY KEY,
            event_id TEXT,
            sport_key TEXT,
            sport_title TEXT,
            home_team TEXT,
            away_team TEXT,
            commence_time TEXT,
            player_name TEXT,
            prop_type TEXT,
            outcome_name TEXT,
            outcome_price REAL,
            outcome_point REAL,
            bookmaker_key TEXT,
            bookmaker_title TEXT,
            scraped_at TEXT
        )
    """)
    
    if 'bookmakers' in data:
        for bookmaker in data['bookmakers']:
            for market in bookmaker.get('markets', []):
                for outcome in market.get('outcomes', []):
                    cursor.execute("""
                        INSERT INTO player_props VALUES 
                        (NULL,?,?,?,?,?,?,?,?,?,?,?,?,?,?)
                    """, (
                        data['id'], data['sport_key'], data['sport_title'],
                        data['home_team'], data['away_team'], data['commence_time'],
                        outcome.get('description', ''), market['key'],
                        outcome.get('name', ''), outcome.get('price'),
                        outcome.get('point'), bookmaker['key'],
                        bookmaker['title'], datetime.now().isoformat()
                    ))
    
    conn.commit()
    conn.close()
    print("SUCCESS: Data saved!")

else:
    print("No events found")
```

Run it:
```
python fetch_data.py
```

### 5. Start dashboard
```
python dashboard.py
```

### 6. Open browser
```
http://localhost:5000
```
